Quem somos

O projeto SIMLE foi criado com o objetivo de ensinar os fundamentos do desenvolvimento web de forma clara, orga izada e acessível.

Perguntas frequentes

O que é o projeto SIMPLE

O SIMPLE é um projeto educacional, criado para ensinar HTML e CSS de forma prática,clara e acessível para iniciantes.

Este site usa JavasCript?

Não. Toda a estrrutura apresentada até aqui utiliza apenas HTML semântico e CSS Básico

Posso evoluir este projeto depois?

Sim. O projeto foi pensado para evoluir futuramente para PHP, formulários funcionais e banco de dados.
